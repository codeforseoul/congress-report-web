ASP.NET 응용 프로그램

축하합니다! 프로젝트를 만들었습니다.

이 응용 프로그램은 다음으로 구성됩니다.

  • 홈, 정보 및 연락처 간의 기본 탐색을 표시하는 샘플 페이지
  • Bootstrap을 사용한 테마 설정
  • 인증(선택된 경우 등록 및 로그인 방법 표시)
  • NuGet을 사용하여 관리하는 ASP.NET 기능

응용 프로그램 사용자 지정

  • ASP.NET MVC 시작
  • 사이트 테마 변경
  • NuGet을 사용하여 더 많은 라이브러리 추가
  • 인증 구성
  • 웹 사이트 사용자에 대한 정보 사용자 지정
  • 소셜 공급자로부터 정보 가져오기
  • ASP.NET Web API를 사용하여 HTTP 서비스 추가
  • Web API 보안 설정
  • ASP.NET SignalR로 실시간 웹 추가
  • Scaffolding을 사용하여 구성 요소 추가
  • Browser Link 로 응용 프로그램 테스트
  • 프로젝트 공유

배포

  • 응용 프로그램이 프로덕션할 준비가 되었는지 확인
  • Windows Azure
  • 호스팅 공급자
